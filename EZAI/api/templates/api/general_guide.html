Submitting a model

Let's get started!

The aim of ezai is to make the task of deploying your models as easy as possible. With a little effort from the developers side, we can make this process painless for everyone. There are a few steps that you have to take.

Accepted models

Currently the ezai platform only supports models made with python, supporting the following libraries:


  • • Sklearn


  • • Keras (TF backend)


  • • Numpy


Donâ€™t worry, the fact that your favorite python library is not listed here does not necessarily mean it wont work. For libraries to get the supported status means that it is tested on our system.

Process

Sklearn and Numpy

Once you have completed your model, it has to be formatted in a way that our platform can handle. This means that you have to serialize it by using a library called pickle. Pickle will serialize your python code into a binary that we can store and run. For more information read: https://docs.python.org/2/library/pickle.html.
